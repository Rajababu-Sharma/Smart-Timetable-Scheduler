# Smart-Timetable-Scheduler
An automated college timetable generation system that creates shift-based, clash-free schedules for departments, faculty, classrooms, and laboratories.

# Overview

Smart Time Table Scheduler is designed to eliminate the complexity and errors involved in manual timetable preparation.

The system intelligently assigns subjects, faculty, and rooms while ensuring:

1.No faculty clashes
2.No classroom conflicts
3.Proper shift allocation 
4.Balanced daily lecture distribution

# Objective

To develop an efficient scheduling system that:
1.Automates timetable generation
2.Reduces administrative workload
3.Prevents overlapping assignments
